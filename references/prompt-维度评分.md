# 六维怎么读（读法 + 90 分锚 / 60 分锚）
- **开局抓力**：读法＝前三章有没有在「读者失去耐心之前」给到第一个爽或钩。90＝三章内首爽兑现＋钩子未解＋代入零门槛；60＝首爽拖到五章后，或前三章还在铺世界观。
- **爽点系统**：读法＝看「铺垫→释放→余波」这条链完不完整、递不递进，**不是数爽点个数**。90＝爽点有铺垫有兑现有余波、逐段加码；60＝爽点稀疏或套路单一、常无铺垫直接爽。
- **追读钩子**：读法＝章末让不让人「手指想点下一章」，盯钩子强度曲线有没有连续塌陷段。90＝章末钩子密且强、跨章悬念叠套；60＝钩子时有时无、断章随意。
- **节奏与留存**：读法＝水章率＋掉读风险章，连续无爽无钩＝读者流失点。90＝水章率极低、无连续掉读段、卷末高潮命中；60＝出现连续低投入段或节奏忽快忽慢。
- **题材卖点**：读法＝一句话能不能说清卖点、够不够当红、和目标平台合不合。90＝题材当红＋卖点一句话说清＋有差异化；60＝题材偏冷或卖点模糊。
- **长线承载**：读法＝世界观 / 金手指够不够撑几百万字、坑埋得深不深。90＝体系可长线展开＋升级路径清晰＋深坑；60＝设定容量有限、长线有隐忧。

# 打分纪律
- **硬约束1（数量≠质量）**：爽点系统 / 追读钩子 两维必须按证据引用里的**执行质量**打分——有无铺垫、是否兑现、余波是否落地；聚合的密度 / 强度曲线只作辅助参考。高密度但普遍无铺垫、无兑现的应判低分，**不得因数量多而给高分**。反之，单个铺垫充分、兑现到位的爽点，可高于一堆平铺直叙的爽点。
- **硬约束2**：每条 hard_issues 必须挂一个 commercial_mechanism；挂不上的（纯道德 / 价值观）不许出现——见立场声明第 4 条。
- 分数别扎堆在 70-80。该 90 就 90、该 40 就 40——你是主编，不是老好人。
